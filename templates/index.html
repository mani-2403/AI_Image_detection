🧠 About the AI Model

This system utilizes the umm-maybe/AI-image-detector, a state-of-the-art Image Classification model deployed via HuggingFace Transformers. It is specifically trained to distinguish between natural (human-captured) and artificial (AI-generated or deepfake) images.

How it works

  • Image Input: The uploaded image is resized and normalized for the neural network.
  • Feature Extraction: Deep Convolutional layers detect unnatural textures and artifacts.
  • Classification: The model computes the probability of the image being "artificial".
  • Hardware Trigger: Results are sent via PySerial to trigger Arduino indicators.

Confidence Score

The AI produces an "artificial" probability score from 0 to 1. If this score exceeds 0.5, the image is flagged as FAKE. Otherwise, it is classified as REAL. The percentage displayed reflects the model's certainty in its prediction.
